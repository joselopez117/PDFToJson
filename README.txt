1. Download the project to your computer
2. Copy and paste your PDF of choice into the files folder (remove the test pdf)
3. Rename the pdf you want scanned to "filetoscan.pdf"
4. Open the project and open a terminal (ctrl + `)
5. Type "npm install" allow for commmand to run do not cancel it
6. Type "node runthis.js"
7. in the output folder, you should have the resulting json!
